# BlackBox GPT — Telegram Bot

Universal AI assistant Telegram-bot with two modes:

- **Universal** — DeepSeek model without web search.
- **Professional** — DeepSeek + Perplexity with web search and premium limits.

Includes:
- CryptoBot-based subscription system (USDT).
- Request limits for free and premium users.
- PostgreSQL storage (users, subscriptions, daily_limits).
