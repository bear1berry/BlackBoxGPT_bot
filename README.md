# BlackBox GPT — Universal AI Assistant

Минималистичный Telegram‑бот с нижним таскбаром (reply‑клавиатура) и несколькими
режимами мышления. Бот умеет работать с DeepSeek и Perplexity Sonar и может
автоматически переключать провайдера в зависимости от режима.

См. `.env.example` и файл `deploy/blackboxgpt.service` для деплоя на Yandex Cloud.
